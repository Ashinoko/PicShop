# PicShop
A prototype Online Shop(for pictures), with Cart and favorite functionalities. made with ReactJS


## Requierments
npm 6.14.17 , 
vite 3.2.3


## Packages
prop-types: 15.8.1,
react-router-dom: 6.5.0


### Running
"npm run dev" on CLI to run on localhost
